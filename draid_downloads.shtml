This is it: The game itself. After a long long waiting period, it has finally come true: You can download DerangedRaid and wittness all the eyecandy with your own eyes.
NOTE: This is a beta-version. Maybe just a demo. It depends on the point of view. You should see it as a "basic package", which is able to update itself automatically via the "AutoUpdater", accessible within the game. Many small and some quite big improvements and new features will eventually find their way to you via these Auto-Updates.
But for now, enjoy what's there. And remember that this really isn't a finished game.




 Deranged Raid RELEASE 1


Features:

Large detailed terrain
5 different terrain types
Realistic day/night cycles
Dynamic Level-Of-Detail
MP3 Soundtrack, can be replaced by an unlimited number of custom MP3s
3 different ships to fly
Many different buildings
Powerful and funky weapons (homing missiles, satellite cannon, etc)
Integrated Auo-Updater provides new features and weapons as they become available

Not contained in this release:

Strategical-AI
Multiplayer
Game Interface
Weather effects
